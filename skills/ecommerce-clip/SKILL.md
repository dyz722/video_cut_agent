# 带货切片剪辑策略

目标: 从直播录屏或口播素材中产出 hook-driven 的带货短视频, 9:16 竖屏 (1080x1920)。

## 一、钩子识别 (决定片子生死)

在 transcript 里用 bash grep 定位以下话术模式, 它们是钩子候选:

- **价格冲击**: "折"/"优惠"/"便宜"/"只要X元"/"半价"/"50%"/"满减"/"to 减"
- **稀缺紧迫**: "最后"/"仅剩"/"今天"/"错过"/"限量"/"秒杀"/"库存"
- **促销机制**: "买二"/"买三"/"送"/"赠"/"加购"/"凑单"/"分期"/"先用后付"
- **效果承诺**: "立刻"/"马上"/"一秒"/"亲测"/"真的"
- **痛点提问**: "是不是"/"有没有"/"还在"开头的疑问句

钩子质量排序: 价格冲击+稀缺 > 促销机制 > 效果承诺 > 痛点提问。
每条片子必须以钩子开场, **钩子句必须出现在 0-3 秒内**。

## 二、结构与节奏

```
0-3s    钩子 (促销话术原声, hook 样式字幕 pop_in 弹入)
3-15s   卖点展示 (商品特写/演示, 必要时 watch_video 确认画面有商品)
15-35s  机制讲解 (价格/赠品/下单方式)
最后3-5s 行动号召 (口播原声"点击下方"类, 或 TTS 补一句)
```

- 总片长 **20-45s**, 超过 45s 砍卖点不砍钩子。
- 单个 clip 不超过 8s, 节奏感靠多 clip 快切; 相邻 clip 之间默认**不加转场**
  (带货要的是快, 转场只在场景大跳变时用 fade 0.3s)。
- 废话/冷场/重复句直接跳过, 用 transcript 时间戳精确切。
- 切点 snap 到 scenes.json 切变点, 避免画面跳动。

## 三、字幕规范

- 钩子句: style=hook + effect=pop_in
- 正文跟读字幕: style=default + effect=none, 一句一条, 与原声时间严格对齐
  (直接用 transcript 的句级时间戳), 每条不超过 15 个字, 长句拆条。
- 价格/数字用 promo 样式单独强调 (如 "第二件0元")。

## 四、促销横幅 (overlays)

- 活动期 (双11/黑五/5.5 Sales 等) 在顶部加品牌横幅: y=80, width=900,
  全程显示或前 10s 显示。
- 横幅素材在 materials/ 里找 banner/活动名相关的 png; 没有就不加, 不要硬造。

## 五、声音

- 保留原声 (volume 1.0), 直播底噪重时可降到 0.9。
- BGM 可选: 节奏感强的轻电子, volume 0.2, ducking=true。
- 结尾行动号召缺原声时用 tts 补:
  voice=longanyang, instruction="你正在进行广告促销，你说话的情感是happy。"

## 六、批量出片

一场直播切 N 条: 先全场 transcribe(background=true) + detect_scenes,
然后按钩子质量排序取 top N, 每条独立 timeline_XX.json。
超过 30 分钟的直播用 task 派 subagent 分段(20min/段)找钩子, 回收
"[start-end] 钩子类型 + 原话" 格式清单后统一排序。

## 七、质检红线

- 开头 3s 内没有钩子字幕 -> 不合格
- 字幕越界/遮挡商品 -> 调 margin_v 重渲染
- 响度低于 -25 LUFS -> 检查 clip volume
- watch_video 确认: 片中至少出现一次商品清晰画面
